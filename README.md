# Usage:
- Create a CSS file.
- Import FontKit using
```css
@import url("https://noahzeisberg.github.io/fontkit/source.css");
```

- Apply Font to everything and add an alternate font
```css
* {
    font-family: "Font Name", sans-serif
}
```
